Project:
Smart Energy Management System using ESP32 and Arduino Cloud

Description:
An IoT-based system that monitors occupancy, temperature, humidity, and ambient light levels to optimize HVAC and lighting energy consumption in commercial environments.

Features:
- Occupancy detection using PIR sensor
- Temperature and humidity monitoring using DHT22
- Ambient light monitoring using LDR
- Cloud-based monitoring through Arduino Cloud
- Automated relay control for HVAC and lighting
- Remote dashboard visualization

Hardware:
- ESP32
- DHT22
- PIR Sensor
- LDR
- Relay Module

Software:
- Arduino IDE
- Arduino Cloud
